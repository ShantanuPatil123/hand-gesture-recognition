# Hand-Gesture-Recognition
A real-time hand gesture detection project built using Python, OpenCV, and MediaPipe. This system identifies gestures such as thumbs up, fist, open palm, and more using computer vision techniques.

# Features
-Real-time gesture detection
-Tracks 21 hand landmarks
-High accuracy and fast processing
-Works on any webcam-enabled device
-Easy to extend for custom gestures

# Tech Stack
-Python
-OpenCV
-MediaPipe
-NumPy
